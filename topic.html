Introduction

This tutorial demonstrates different possibility to use the JavaScript toolkit of Verovio in a web environment. It goes from the most basic case to more complicated ones that make use of the interaction capabilities of the toolkit.

The ZIP file with the tutorial can be downloaded from here.

With some browsers, like Firefox, you can run it by opening the HTML file in the browser.

With other browsers, you might need to run a server : in the terminal, go to the folder where the HTML file is and type the following command : python -m SimpleHTTPServer 8003. The file will then be available on http://localhost:8003.
