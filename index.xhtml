Verovio is a fast, portable and lightweight library for engraving Music Encoding Initiative (MEI) music scores into SVG. It is designed as a tool usable as a one-step conversion tool and it is written in pure standard C++ with no dependencies on third-party frameworks and libraries. Verovio implements its own rendering engine, which can render SVG with all the musical symbols embedded in it and without dependency on external fonts. In addition to MEI, Verovio can also render Plain and Easy (PAE) code and DARMS code.

Since the rendering engine is defined as an abstract component, Verovio is highly modular and can be used in different contexts, including as a JavaScript toolkit compiled with Emscripten. It is possible to implement easily a rendering output different from SVG if needed.

Verovio source code is available on GitHub at https://github.com/rism-ch/verovio

Verovio is developed by the Swiss RISM Office and is supported by the Swiss National Science Foundation. It is available under the GPLv3 license.

Why Verovio?

Verovio is named after a Dutch music engraver, Simone Verovio, who was active in Rome in the late sixteenth century. He was one of the very first to use engraving on copper plates for printing music, a technique that was revealed to be the most suited to music about one century later.
