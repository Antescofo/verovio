Source code

Verovio source code is available on GitHub at https://github.com/rism-ch/verovio

To obtain the current version (master), run:

{% highlight bash %}$ git clone https://github.com/rism-ch/verovio.git -b master verovio{% endhighlight %}

JavaScript toolkit

The GitHub repository also provides a compiled version of the JavaScript toolkit for each release at https://github.com/rism-ch/verovio/releases

Instructions for using the JavaScript toolkit are given on the toolkit page.

Building the command-line tool

To build the command-line tool with CMake, run:

{% highlight bash %} $ cd tools $ cmake . $ make $ sudo make install {% endhighlight %}

For seeing the command-line options, run:

{% highlight bash %}$ ./verovio --help{% endhighlight %}
Building with XCode

For OS X users, there is also an XCode 4 project in the Verovio root directory.

Building the JavaScript toolkit

To build the JavaScript toolkit you need to have the Emscripten compiler installed on your machine. Then run:

{% highlight bash %} $ cd emscripten $ ./build.sh {% endhighlight %}

The toolkit will be written in:

{% highlight bash %}./emscripten/build/verovio-toolkit.js{% endhighlight %}

Building the Python toolkit

To build the Python toolkit you need to have swig and swig-python installed on your machine (see SWIG) and the Python distutils package. Then, to build it in-place, run:

{% highlight bash %} $ cd python $ python setup.py build_ext --inplace {% endhighlight %}

If you want to install it, run:

{% highlight bash %} $ cd python $ python setup.py build_ext $ sudo python setup.py install {% endhighlight %}
