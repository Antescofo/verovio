Highlighting content

This example shows how Verovio can access MEI elements for interaction. With the jQuery-Selector $("xx") all xx-Elements inside a MEI file are selected, so $("dynam") selects all <dynam> elements in a file. You can select a whole class $(".class") or even a single element by using $("#id").

{% highlight javascript %} $(".dynam" ).attr("fill", "#d00").attr("stroke", "#d00"); {% endhighlight %} {% highlight javascript %} $(".measure .staff:nth-child(2)" ).attr("fill", "#00e").attr("stroke", "#00e"); {% endhighlight %} {% highlight javascript %} $(".measure .staff:nth-child(4) .staffDef" ).attr("fill", "#0c0").attr("stroke", "#0c0"); {% endhighlight %} {% highlight javascript %} $(".system .measure:gt(4) .staff:nth-child(4)").attr("fill", "#ccc").attr("stroke", "#ccc"); {% endhighlight %}
