Accessing MEI elements

This example shows how Verovio can access MEI elements for interaction. With the jQuery-Selector $("xx") all xx-Elements inside a MEI file are selected, so $("dynam") selects all elements in a file. You can select a whole class $(".class") or even a single element by using $("#id").

{% highlight javascript %} ///////////////////////////// /* An id in the file below */ ///////////////////////////// id = "chord-L1186F7"; {% endhighlight %}

{% highlight javascript %} ////////////////////////////////////////////////////////////////////// /* A function that loads the page where the id is and highlights it */ ////////////////////////////////////////////////////////////////////// function loadPage() { var elementPage = vrvToolkit.getPageWithElement(id); if (elementPage == 0) { console.log("ID not found"); } else { page = elementPage; } svg = vrvToolkit.renderPage(page, ""); $("#svg_output").html(svg); /////////////////////////// /* Highlight the element */ /////////////////////////// $("#" + id).attr("fill", "#f00").attr("stroke", "#f00"); }; {% endhighlight %}
