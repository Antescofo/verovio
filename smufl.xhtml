Verovio follows the Standard Music Font Layout (SMuFL) specification. It means that it is easily possible to change the music font used in Verovio and to have a personalised output. Verovio includes its own SMuFL-compliant music font, Leipzig. It also supports and includes the Bravura designed by Daniel Spreadbury at Steinberg.

The font can be selected using the --font option in the command-line tool or by adding { font: "Bravura" } in the JavaScript toolkit options.
