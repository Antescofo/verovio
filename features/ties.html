Ties

Verovio supports both ties encoded with @tie attributes or with <tie> elements. Ties overlapping two measures and interrupted by a system break or a page break are displayed appropriately.

{% include mei-excerpt.html id="code-tie" file="tie.mei" %}
{% highlight xml %} {% include features/tie.txt %} {% endhighlight %}

Verovio determines the tie direction looking at the context (layer, stem direction, chord, etc.). Collision with stem is normally avoided except with short ties and when the context does allow do to so.

{% highlight xml %} {% include features/tie-chords1.txt %} {% endhighlight %}

With chords, the @tie attribute can be set in the chord or in the notes. It can also be overwritten in the note if necessary.
