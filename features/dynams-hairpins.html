Dynamics

Verovio supports <dynam> elements. Horizontal positioning can be specified using either the @startid or @tstamp attribute.

When the text of the dynamic has corresponding code in SMUFL, the music font will be used. For example, for <dynam>p</dynam>, the SMuFL character U+E520 (dynamicPiano) will be used. Dynamics with corresponding SMuFL code are centered under the horizontal alignment points.

Placement and scale

By default, dynamics are placed below the staff. They can be placed above using the @place attribute. Dynamics are also scaled according to the @scale given in staffDef.

Hairpins

Verovio supports <hairpin> elements. As for dynamics the placement can be specified with the @place attribute. Hairpins overlapping two measures or more and interrupted by a system break or a page break are displayed appropriately.
