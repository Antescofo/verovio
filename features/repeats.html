Repeats

Verovio supports <mRpt>. The number on top of each repeat is calculated by Verovio. The @mutli.number in <scoreDef> or <staffDef> can be set to false for not showing the number. (The value of the @mutli.number will be set for the entire score or staff and cannot be redefined.)

Beat repeats

Verovio also supports beat repetition with <beatRpt> elements. The value of @rend determines the number of slashes, including for @rend="mixed" The alignment of the beat repetition will not work with compound meters when the repeated beat duration do not correspond to the meter signature unit.
