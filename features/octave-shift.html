Octave shift

Verovio supports <oct> for octave shifts. For correct MIDI output, the @oct.ges attribute needs to be provided.

For ottava bassa displacements the @place attribute has to have the value below. The quindicesima displacement is also supported. The scope of the octave shift can be specified either with @startid / @endid or with @tstamp / @tstamp2 attributes.
