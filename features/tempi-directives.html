Tempi

Verovio supports <tempo> elements. Horizontal positioning can be specified using either the @startid or @tstamp attribute. By default, tempi indications are placed above the staff.

The <rend> element can used within the text in order to include SMuFL codes, for example for specifying metronome values. The @fontname attribute needs to have the value VerovioText. (Only note SMuFL codes are included in the VerovioText font for now.)

Typographic formatting

By default, tempi indications are rendered in bold. This can be modified by including a <rend> element in the text. It is also possible to have nested <rend> elements.

Directives

Verovio supports <dir> elements. Horizontal positioning can be specified using either the @startid or @tstamp attribute. By default, directive indications are placed below the staff and are rendered in italic.

{% include mei-excerpt.html id="code3" file="dir1.mei" %}
{% highlight xml %} {% include features/dir1.txt %} {% endhighlight %}

Typographic formatting within directive elements is supported as with tempi (see above). As with hairpin, a single directive element can be applied to multiple staves. Editorial markup is also supported within directives.
